# EasyGo
Decentralization Taxi Platform 0% commisions 
EasyGo — децентрализованная такси-платформа кооперативного типа, построенная на модели совладения водителей. Платформа устраняет посредника-агрегатора: комиссия 0% навсегда, а сэкономленные средства за счет 0% комиссии  перераспределяются между участниками поездки (60% водителю / 40% пассажиру). Монетизация строится на экосистеме товаров и услуг для автопредпринимателей (топливо, страховка, ТО, аренда,услуги..), а не на поездках и комиссии водителей.

Целевые пользователи:

Автопредприниматели (водители-ИП/самозанятые) — ищут выгодную альтернативу агрегаторам с сохранением дохода и обретением статуса совладельца компани "EasyGo"
Пассажиры — получают поездки дешевле за счёт 0% комиссии агрегатора, прямую связь с водителем, возможность выбирать избранного водителя и осознанный выбор ("деньги остаются водителю, не корпорации")
Основатель / администраторы — управляют кооперативом через мастер-панель


Что делает EasyGo особенным

Ключевой инсайт: Водители платят одни и те же операционные расходы в любом агрегаторе — разница лишь в том, кому достаётся комиссия. EasyGo возвращает эту комиссию участникам.

Дифференциаторы:

0% комиссии навсегда — не акция, это бизнес-модель
Личный кабинет автопредпринимателя — реальная себестоимость поездки, чистая прибыль, KPI, кооперативная доля. Прозрачность которой нет ни у одного агрегатора

Самоуправление сообщества (Фаза 2) — блокировки, правила и тарифная политика принимаются голосованием водителей через Совет сообщества автопредпринимателей, не корпоративным алгоритмом
Цифровой Пай (Фаза 2) — каждый водитель получает долю прибыли платформы пропорционально вкладу и kpi автопредпринимателя.
Реферальная программа — 2% с поездок за каждого приглашённого водителя и пассажира (только деньгами, не баллами); пассажир получает скидку 30% на первые 10 поездок при регистрации по реферальному коду водителя
Приветственный топливный бонус — 1 000 ₽ при регистрации как автопредприниматель
Стратегия нулевого риска — водитель не уходит из текущего агрегатора: "Попробуй параллельно — 2-3 заказа в день без комиссии. Посмотри сам."
Открытая финансовая книга — все доходы и расходы платформы публичны в реальном времени. Прозрачность как конкурентное оружие.
Слоган для водителей: "Водитель перестаёт быть рабочим — он становится совладельцем платформы EasyGo."

Слоган для пассажиров: "Твой водитель. Без посредников. Дешевле."
