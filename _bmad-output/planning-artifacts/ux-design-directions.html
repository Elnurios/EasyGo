для EasyGo

Каждое — полноценная визуальная система с дашбордом водителя, реферальным кодом и экраном Избранных. Одни и те же данные, разный характер.

// направление 01
NODE — Узел сети
Метафора децентрализованной сети. Водитель — узел. Реферальный код — access token. Кибер-акценты точечно: свечение, терминальный шрифт, сетчатый фон едва заметен. 90% чистота, 10% характер.
+ Чёткая метафора + Кибер-дух без перегруза + Данные читаются легко ~ Может показаться технически

// sally.recommendation
Рекомендация: NODE как база + элементы PULSE
NODE даёт чёткость и читаемость — критично для водителя который смотрит на числа в машине. PULSE-элементы добавить точечно: кольцо-индикатор на главном экране, анимация boot-sequence при регистрации, более выраженный glow на реферальном коде. Так соблюдается 10% правило и сохраняется ДНК платформы.
// карта маршрута
Карта 2ГИС — тёмный стиль EasyGo
Карта маршрута интегрирована в приложение водителя и пассажира. Стиль карты адаптирован под тёмную тему EasyGo: тёмные улицы, фиолетовый маршрут, минимум визуального шума. Точка A — подача, точка Б — назначение.
+ 2ГИС тёмный стиль + Маршрут A→Б + Позиция водителя live + ETA и дистанция
