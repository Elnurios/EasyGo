# Стратегия ценностного предложения для пассажира EasyGo

## Главный вопрос

> **Что предложить пассажиру, чтобы он пользовался EasyGo?**

---

## Три якоря ценности

Все шесть агентов сошлись на трёх уровнях, которые вместе образуют полную стратегию:

| Якорь | Тип | Сообщение |
|-------|-----|-----------|
| **Предсказуемая цена** | Рациональный | "Честная цена — всегда. Без сюрпризов." |
| **Любимый водитель** | Эмоциональный | "Твой водитель. Бронируй напрямую." |
| **Открытая финансовая книга** | Идеологический | "Смотри сам, куда идут твои деньги." |

---

## Якорь 1 — Рациональный: Предсказуемая цена

**Почему работает:**
- Яндекс Go использует surge-pricing (x2–3 в пятницу вечером, в дождь, после концертов)
- Пассажир не понимает почему цена меняется — это вызывает недоверие и раздражение
- 0% комиссии даёт EasyGo экономическую возможность держать цену стабильной

**Как реализовать в UI:**
- Бейдж рядом с ценой: *"Фиксированная цена. Не меняется в пути."*
- Сравнение в момент заказа: *"Яндекс сейчас: 350 ₽ → EasyGo: 280 ₽"*
- После поездки: *"Вы сэкономили 87 ₽ сегодня."*


**Триггер переключения (гипотеза John):**
Пассажир переключается при видимой разнице ≥ 15% прямо в UI.

---

## Якорь 2 — Эмоциональный: Любимый водитель

**Почему работает:**
- Uber (2019): пассажиры с предпочтительным водителем делают на **34% больше поездок**
- Яндекс Go — лотерея каждый раз. EasyGo даёт "своего" человека в городе
- Целевые сегменты: мамы с детьми, корпоративные клиенты, пожилые пассажиры

**Механика:**
- После завершённой поездки — предложение добавить в избранное
- После 3-й поездки с одним водителем — мягкий nudge: *"Похоже, вы нашли своего водителя. Сохранить?"*
- Прямое бронирование: *"Антон свободен через 12 минут — забронировать?"*
- Профиль водителя: фото, рейтинг, количество совместных поездок, стиль езды

**Нарратив:**
> *"Алексей получает 100% от поездки. Никакой комиссии. Ты платишь меньше — он зарабатывает больше."*

**Гипотеза (John):**
Функция "Любимый водитель" увеличивает retention через 30 дней в 2+ раза по сравнению с обычным пользователем.

---

## Якорь 3 — Идеологический: Открытая финансовая книга

**Почему работает:**
- 64% потребителей лоялят бренду, разделяющему их ценности (Edelman Trust Barometer)
- Ни один агрегатор в мире не публикует свои финансы в реальном времени
- Формирует новый тип пользователя — не просто клиент, а **сторонник**

**Как реализовать в UI:**
- Экран *"Куда пошли твои деньги"* после каждой поездки: `340 ₽ → водителю | 0 ₽ → комиссии`
- Публичный дашборд (статика, обновление раз в минуту): *"Сегодня водители кооператива заработали 2 847 300 ₽"*
- Ссылка на открытую книгу прямо в приложении: *"Смотри сам."*

**Техническая оценка (Winston):**
Публичный дашборд — read-only витрина над БД, статика раз в минуту. Реализация: **1–2 дня.** Без real-time WebSocket в MVP.

## Воронка удержания пассажира

```
Первая поездка
└── Триггер: 30% скидка на первые 10 поездок (по реферальному коду водителя)

Повторная поездка
└── Триггер: сравнение цены с Яндексом + предложение "Любимого водителя"

Привычка (5–10 поездок)
└── Триггер: реферальный кэшбэк 2% деньгами навсегда

Адвокат бренда
└── Триггер: Открытая книга, гордость за выбор ("я помогаю водителю")
```

## Приоритеты фич (MVP)

**Fastest path to value (Amelia):**
```
referral → discount → favorite_driver
```

| Фича | Приоритет | Сложность | Статус в PRD |
|------|-----------|-----------|--------------|
| 30% скидка на первые 10 поездок | 🔴 Критично | Низкая | FR43–FR45 |
| Реферальный кэшбэк 2% | 🔴 Критично | Низкая | FR18–FR21 |
| Любимый водитель | 🔴 Критично | Средняя | FR46–FR52 |
| История поездок | 🟡 Важно | Низкая | — нет в PRD |
| Профиль водителя до посадки | 🟡 Важно | Низкая | — нет в PRD |
| Публичный финансовый дашборд | 🟡 Важно | Низкая (1-2 дня) | FR42 |
| Сравнение цены с Яндексом в UI | 🟡 Важно | Низкая | — нет в PRD |

**Технические ограничения (Winston, Amelia):**
- Кастомное пассажирское приложение — **нет**, TaxiMaster владеет UI
- 30% скидка до оплаты предпочтительнее кэшбэка психологически; реализуемость зависит от TaxiMaster API
- Push-уведомления ограничены TaxiMaster SDK

## Мессенджинг — что говорить пассажиру

### Первые 10 секунд (один удар)

> **"Те же машины. Те же водители. Дешевле — потому что без Яндекса."**

### Слова, которые работают ✅

| Фраза | Почему |
|-------|--------|
| "Без посредников" | Понятно и злободневно |
| "Твой водитель" | Личное, тёплое |
| "Честная цена" | Не "низкая", а *справедливая* |
| "Водитель зарабатывает больше" | Пассажир чувствует себя хорошим человеком |
| "Фиксированная цена. Не меняется в пути." | Снимает тревогу surge-pricing |

### Слова под запретом ❌

| Слово | Почему |
|-------|--------|
| "Децентрализованный" | Никто не поймёт |
| "Кооператив" | Звучит как СССР |
| "Экосистема" | Корпоративный новояз |
| "Инновационная платформа" | Штамп |

### Три уровня коммуникации

1. **Эмоциональный** (баннер, слоган): *"Водитель получает всё. Ты платишь меньше."*
2. **Рациональный** (онбординг, 2–3 слайда): сравнительная таблица Яндекс vs EasyGo
3. **Доверие** (профиль, открытые финансы): *"Смотри сам: все расходы платформы открыты."*
