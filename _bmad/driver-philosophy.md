# Философия и экономика автопредпринимателя EasyGo

## Часть 1 — Философия: ты — автопредприниматель и совладелец

В EasyGo водитель — **партнёр платформы**, а не наёмный исполнитель:

- Ты знаешь **реальную себестоимость** каждой своей поездки
- Ты видишь **прозрачный расчёт** своей прибыли с каждого рубля
- Платформа мониторит рынок и предлагает **максимально выгодную цену**
  для пассажира, опираясь на твою себестоимость
- Ты не гонишься за коэффициентами — ты работаешь по **стабильной, честной экономике**
- Ты — совладелец платформы, которая работает на тебя

> **Принцип EasyGo:** не максимальный коэффициент, а максимальная чистая прибыль
> при минимальных издержках и предсказуемом доходе.

### Почему не стоит гоняться за повышенными коэффициентами

Высокий коэффициент выглядит привлекательно, но средняя чистая прибыль
за смену всегда одинакова — рынок сам себя балансирует.
Гоняясь за коэффициентами, водитель тратит больше топлива,
изнашивает машину быстрее и зарабатывает столько же — только устаёт сильнее.

EasyGo даёт стабильный доход без погони за пиками.

## Часть 2 — Полная себестоимость поездки автопредпринимателя

### Состав издержек на 1 км

| Статья | Входит в расчёт |
| ------ | --------------- |
| Топливо | Да |
| Техническое обслуживание (масло, фильтры, тормоза, подвеска) | Да |
| Аренда автомобиля (если нет своего) | Да |
| Амортизация автомобиля | Да |
| Штрафы | Да |
| Страховка (ОСАГО / КАСКО) | Да |
| Автомойка | Да |
| Налог (самозанятый / ИП, ~4–6% от дохода) | Да |

---

### Два профиля автопредпринимателя

| Профиль | Издержки на км | Комментарий |
| ------- | -------------- | ----------- |
| **Своё авто** | **12 ₽/км** | Нет арендной платы, есть амортизация |
| **Аренда авто** | **17 ₽/км** | Аренда включена, амортизация — у арендодателя |

> Источник: средние показатели водителей-автопредпринимателей (Ростов-на-Дону).

---

### Себестоимость поездки (формула)

```text
СЕБЕСТОИМОСТЬ ПОЕЗДКИ =
  Расстояние (км) × Ставка профиля (12 или 17 ₽/км)
  + Время поездки (мин) × 11,7 ₽/мин
```

**Стоимость 1 минуты рабочего времени:**

- Целевой минимум: **700 ₽/час грязными**
- 700 ₽ ÷ 60 мин = **~11,7 ₽/мин**

---

### Пример расчёта себестоимости: поездка 10 км, 15 минут

| Статья | Своё авто (12 ₽/км) | Аренда авто (17 ₽/км) |
| ------ | ------------------- | --------------------- |
| Издержки на км | 10 × 12 = 120 ₽ | 10 × 17 = 170 ₽ |
| Стоимость времени | 15 × 11,7 = 175 ₽ | 15 × 11,7 = 175 ₽ |
| **Себестоимость** | **295 ₽** | **345 ₽** |

---

### Пример расчёта прибыли в EasyGo

| Статья | Своё авто | Аренда авто |
| ------ | --------- | ----------- |
| Себестоимость | 295 ₽ | 345 ₽ |
| Цена для пассажира | **380 ₽** | **430 ₽** |
| Комиссия EasyGo | 0 ₽ | 0 ₽ |
| Водитель получает | 380 ₽ | 430 ₽ |
| **Чистая прибыль** | **+85 ₽** | **+85 ₽** |

> EasyGo рассчитывает цену **от реальной себестоимости профиля водителя** —
> водитель всегда в плюсе.

## Часть 3 — Логика ценообразования EasyGo

### Принцип: себестоимость + мониторинг рынка

```text
ЦЕНА ПОЕЗДКИ В EasyGo =
  Себестоимость поездки
  + Минимальная прибыль водителя
  + Корректировка по рынку (конкурентная цена)
```

**Алгоритм платформы:**

1. Рассчитать себестоимость маршрута A→B для конкретного автопредпринимателя
2. Мониторить цену на тот же маршрут у конкурентов
3. Предложить цену: **конкурентную для пассажира** и **выше себестоимости для водителя**
4. Водитель получает **100% цены поездки** — без комиссии
