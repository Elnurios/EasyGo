# Бизнес-модель EasyGo

## Концепция

EasyGo — такси-платформа кооперативного типа, где **комиссия с поездок всегда 0%**.
Платформа зарабатывает через экосистему товаров и услуг для автопредпринимателей,
а не за счёт водителей и пассажиров.

---

## Модель распределения стоимости поездки

### Принцип

Комиссия, которую берут традиционные агрегаторы (например, 25%),
в EasyGo распределяется между участниками поездки: **60% водителю / 40% пассажиру**.

### Пример расчёта (экономия от отсутствия комиссии 25%)

| Показатель | Традиционный агрегатор | EasyGo | Разница |
| ---------- | ---------------------- | ------ | ------- |
| Цена поездки для пассажира | 500 ₽ | 450 ₽ | −50 ₽ (−10%) |
| Комиссия платформе | 125 ₽ | 0 ₽ | — |
| Доход водителя | 375 ₽ | 450 ₽ | +75 ₽ (+20%) |
| Перераспределено | — | 125 ₽ | пасс. 40% / вод. 60% |

**Итог:** деньги остаются у участников поездки.

---

## Источники дохода платформы

### 1. Экосистема товаров и услуг для автопредпринимателей

Водители в любом случае несут повседневные издержки — EasyGo поставляет
эти продукты и услуги на эксклюзивных условиях по лучшей цене.

| Категория | Описание |
| --------- | -------- |
| Топливо | Партнёрские АЗС, скидки и кэшбэк |
| Страховка | ОСАГО/КАСКО через платформу |
| Аренда автомобилей | Без наценки таксопарков |
| Техническое обслуживание | Сервисные партнёры со скидкой |
| Автомойка | Сеть партнёрских моек |
| Прочие издержки | Шины, аксессуары, запчасти |

> Логика: работаешь без комиссии и получаешь долю прибыли платформы —
> пользуешься товарами и услугами EasyGo. Это партнёрство внутри кооператива,
> а не принуждение.

### 2. Рекламные доходы

Платформа привлекает рекламодателей. Доход от рекламы распределяется
между водителями как дополнительная пассивная прибыль.

### 3. Реферальная программа (самоокупаемый рост)

Водители и пассажиры сами привлекают новых участников.

| Параметр | Значение |
| -------- | -------- |
| Реферальный % | **2% с поездок** |
| За водителя | 2% с каждой поездки приглашённого водителя |
| За пассажира | 2% с каждой поездки приглашённого пассажира |
| Форма выплаты | Денежный эквивалент на счёт водителя |

## Приветственный бонус при регистрации

**Топливный бонус — 1 000 ₽** — разовый подарок при регистрации как **автопредприниматель**.
Это не постоянная субсидия и не маркетинговая акция — это жест партнёрства
при входе в кооператив.

## Ценностное предложение по сторонам

### Водитель

- 0% комиссия навсегда → +20% к доходу с каждой поездки
- Доля прибыли платформы ежемесячно
- Рекламный доход (пассивно)
- 2% реферальных с приглашённых участников
- Эксклюзивные цены на топливо, страховку, ТО, аренду
- Приветственный топливный бонус 1 000 ₽ при старте
- Совладение и голос в Совете водителей

### Пассажир

- Поездки дешевле на ~10%
- Осознанный выбор: деньги остаются водителю, не корпорации
- 2% реферальных за приглашённых пассажиров

### Платформа EasyGo

- Маржа с экосистемы услуг (не с поездок)
- Рекламные контракты
- Органический рост через реферальную сеть
- Лояльное сообщество совладельцев

## Конкурентное преимущество

| Фактор | Традиционные агрегаторы | EasyGo |
| ------ | ----------------------- | ------ |
| Комиссия | 25–35% | **0% всегда** |
| Совладение водителя | Нет | **Да (кооператив)** |
| Доля прибыли | Нет | **Да (ежемесячно)** |
| Голос водителя | Нет | **Да (Совет)** |
| Цена для пассажира | Рыночная | **Ниже на ~10%** |
| Реферальная выплата | Бонусы/баллы | **Деньги на счёт** |
