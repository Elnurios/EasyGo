# Мастер Админ-панель EasyGo — Панель основателя

## Назначение

Централизованный инструмент управления всей платформой EasyGo.
Доступ только у основателя и доверенных администраторов.

---

## Зона 1 — Главный дашборд (реальное время)

| Показатель | Описание |
| ---------- | -------- |
| Водители онлайн | Количество активных автопредпринимателей прямо сейчас |
| Заказы в работе | Поездки в реальном времени |
| Выручка сегодня | Общая сумма по всем поездкам |
| Новые регистрации | Водители и пассажиры за день / неделю |
| Статус платформы | Работоспособность интеграции TaxiMaster |

---

## Зона 2 — Управление автопредпринимателями

| Функция | Описание |
| ------- | -------- |
| Реестр водителей | Полный список с профилями, контактами, документами |
| Рейтинги | Рейтинг каждого автопредпринимателя, динамика |
| Статусы | Активен / приостановлен / заблокирован |
| История нарушений | Жалобы, предупреждения, решения сообщества |
| Профиль | Своё авто (12 ₽/км) или аренда (17 ₽/км) |
| Управление пассажирами | Реестр, рейтинги, статусы, жалобы |

---

## Зона 3 — Финансовый центр

| Показатель | Описание |
| ---------- | -------- |
| Выручка платформы | По дням / месяцам / периодам |
| Доходы экосистемы | Услуги, партнёрства, реклама |
| Распределение прибыли кооператива | Начислено водителям, периоды выплат |
| Реферальные начисления | Сумма 2% начислений по платформе |
| Приветственные бонусы | Выданные топливные бонусы 1 000 ₽ |

---

## Зона 4 — Управление сообществом

| Функция | Описание |
| ------- | -------- |
| Активные голосования | Открытые вопросы на рассмотрении |
| История решений | Архив принятых решений сообщества |
| Совет водителей | Состав, полномочия, активность |
| Жалобы и апелляции | Входящие обращения, статус рассмотрения |
| Блокировки | Заблокированные водители и пассажиры, причины |

---

## Зона 5 — Интеграция TaxiMaster

| Показатель | Описание |
| ---------- | -------- |
| Статус синхронизации | Работает / ошибка / последнее обновление |
| Ошибки API | Лог ошибок запросов к TaxiMaster |
| Поток данных | Количество запросов, частота polling |
| Данные поездок | Синхронизированные заказы и статистика |

---

## Зона 6 — Аналитика и рынок

| Показатель | Описание |
| ---------- | -------- |
| Мониторинг цен конкурентов | Цены на рынке по популярным маршрутам |
| Зоны спроса | Карта активности заказов по городу |
| Рост сообщества | Динамика регистраций водителей и пассажиров |
| KPI платформы | Средний доход водителя, выполнение заказов, отказы |
| Реферальная воронка | Эффективность реферальной программы |

---

## Принцип доступа

- Мастер-панель видит **всё** — данные водителей, финансы, голосования
- Личный кабинет водителя видит **только своё** — свою статистику и кооперативные данные
- Разграничение через роли: `founder` / `admin` / `driver`
