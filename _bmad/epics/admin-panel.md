# Эпик: Админ-панель управления

## Описание
Веб-панель для управления кооперативом: водители, заказы, финансы, голосования. Разработка с нуля.

## User Stories
- Как админ, я хочу видеть список водителей, чтобы управлять ими.
- Как водитель, я хочу голосовать за решения, чтобы участвовать в управлении.

## Критерии приёмки
- Регистрация/верификация водителей.
- Отчёты по заказам и доходам.
- Система голосования (простая, без блокчейна).

## Зависимости
- Архитектура админ-панели (технический дизайн).
