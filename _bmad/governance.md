# Принципы самоуправления EasyGo

## Главная идея

В традиционных агрегаторах водитель — **исполнитель** под управлением корпорации.
В EasyGo водитель — **автопредприниматель**, который сам управляет платформой.

> **"Водитель перестаёт быть рабочим — он становится совладельцем платформы EasyGo."**

---

## Что изменяется для водителя

| Было (агрегатор) | Стало (EasyGo) |
| ---------------- | -------------- |
| Правила диктует корпорация | Правила принимает сообщество водителей |
| Жалобы уходят в никуда | Решения принимаются голосованием |
| Клиент всегда прав — водитель бессилен | Водители защищают друг друга и себя |
| Блокировка без объяснений | Блокировка только через решение сообщества |
| Техподдержка недоступна | Сообщество помогает сообществу |
| Оценка несправедлива — нет апелляции | Спорные оценки выносятся на рассмотрение |

---

## Система репутации и рейтинга

### Рейтинг автопредпринимателя

- Каждый автопредприниматель видит **свой рейтинг** в личном кабинете
- Рейтинг формируется из оценок пассажиров, статистики поездок и соблюдения правил
- Рейтинги **всех автопредпринимателей открыты** внутри сообщества — прозрачность
- Высокий рейтинг даёт приоритет в распределении заказов и большую долю прибыли

### Рейтинг пассажира

- Пассажиры тоже имеют рейтинг
- Автопредприниматели видят рейтинг пассажира перед принятием заказа
- Пассажир с низким рейтингом может получить отказ без объяснений

## Механизм голосования сообщества

### Кто голосует

Все активные автопредприниматели платформы — пайщики кооператива EasyGo.

### По каким вопросам голосуют

| Вопрос | Кворум |
| ------ | ------ |
| Блокировка автопредпринимателя | Большинство голосов |
| Блокировка пассажира | Большинство голосов |
| Изменение правил платформы | Большинство голосов |
| Тарифная политика | Совет водителей |
| Распределение прибыли | Совет водителей |
| Выбор партнёров экосистемы | Совет водителей |

### Процесс рассмотрения нарушения

```text
Инцидент (жалоба водителя или пассажира)
      │
      ▼
Автоматическая фиксация в системе
      │
      ▼
Рассмотрение Советом водителей
      │
      ├──→ Предупреждение
      ├──→ Временная блокировка
      └──→ Постоянная блокировка (голосование сообщества)
```

## Совет водителей

Совет водителей — выборный орган управления платформой EasyGo.

### Полномочия

- Установка и изменение правил платформы
- Рассмотрение спорных ситуаций и жалоб
- Управление тарифной политикой
- Решения по партнёрствам и экосистеме услуг
- Утверждение распределения прибыли

### Формирование

- Водители с высоким рейтингом и стажем на платформе
- Избираются голосованием всего сообщества
- Ротация состава (например, раз в 6 месяцев)

---

## Защита автопредпринимателя

EasyGo гарантирует водителю:

- **Прозрачность оценок** — водитель видит за что получил оценку
- **Право на апелляцию** — спорная оценка выносится на рассмотрение
- **Защита от необоснованных жалоб** — пассажир с историей ложных жалоб получает сниженный рейтинг
- **Блокировка только по решению сообщества** — никакого произвола алгоритмов
- **Право голоса** — каждый автопредприниматель участвует в управлении платформой
